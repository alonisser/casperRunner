The casperjs html test runner

The wait is over

Casperjs is an awesome project. It enables us to run the headless phantomjs browser(or its Gecko slimerjs companian

Please notice this is based on the experimental FileSytem Api. currently works only with chrom

If you use this runner localy (eg: file:///) and not using a webserver you may need to enable the --allow-file-access-from-files flag when running the chrome binary. or just run this using an localserver (eg: apachs, wamp on windows, Whatever)

Setup:

You need npm (implying Node.js), casperjs and phantomjs installed for this to work

From the command line (in the project folder):

npm install grunt-cli -g npm install grunt watch

Please check Tests are running from the command line before trying the experimental Html test runner

Notice you need to setup the files to test in thefiles key in the casperjs task if not in the default location

refresh to run the test suite
